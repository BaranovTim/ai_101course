Understanding how neural networks make decisions through non-linear transformations.

Key Concept

Without activation functions, a neural network is just a linear regression model. They introduce non-linearity, allowing the network to learn complex patterns and make distinct decisions.

Python - ReLU Implementation 
def relu(x):
    return max(0, x)

# Applying to an array
output = [relu(val) for val in inputs]

Common Functions Overview

In modern deep learning, the choice of activation function can significantly impact the training speed and final performance of your model. Below are the three most critical functions you will encounter.

  • show_chart

    Sigmoid

    Squashes values between 0 and 1. Historically popular but suffers from the vanishing gradient problem in deep networks. Best used for binary classification output layers.

  • timeline

    ReLU (Rectified Linear Unit)

    Outputs x if x > 0, else 0. The default choice for hidden layers due to its computational efficiency and resistance to vanishing gradients.
